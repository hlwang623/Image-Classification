鼓励大家做的事

每类只有 50 张样本，从零训练 ResNet 之类的小模型很容易过拟合。

本次作业的真正学习目标是让大家学会站在大模型的肩膀上解决数据稀缺问题。请大胆探索以下方向（任选其一或组合使用）：

预训练大模型微调：用 ImageNet/LAION 上预训练的视觉模型（ViT、ConvNeXt、DINOv2、EVA 等）作为 backbone，搭配 LoRA、Linear probe、Adapter 等参数高效微调方法在 250 张样本上微调。

CLIP 类视觉-语言模型做 zero-shot / few-shot：自行设计每个 Class_X 对应的文本 prompt，用 CLIP / SigLIP / OpenCLIP 等做 zero-shot 分类；也可以将训练集 50 张作为 support set 做 prompt tuning、线性头训练或最近邻匹配。

大语言模型 + RAG 的 few-shot 推理：用视觉编码器把图像转成 embedding 或描述，把训练样本作为检索库，让 DeepSeek、Qwen、GPT 等大语言模型基于检索到的相似样本做 few-shot 推理。

数据增强 / 自监督 / 伪标签：MixUp、CutMix、RandAugment、SimCLR 风格自监督预训练等都是允许的，只要不引入外部带标签数据。


允许使用任何公开预训练权重和开源工具链（HuggingFace、timm、open_clip、PEFT、LangChain 等），但禁止使用任何外部带标签的图像数据集作为训练或验证。使用预训练模型时请在报告中说明来源与许可证。

几点建议

50 张样本下，特征工程和 prompt 设计往往比模型大小更重要，建议先从 zero-shot 或 linear probe 这种最轻量的 baseline 开始，再逐步加复杂度

测试集很大且不均衡，类别召回率比整体准确率更值得关注

训练集太小，强烈建议留出几张做内部交叉验证，不要盲调超参
