# SpineUS 参赛作品报告（严格模板对齐终稿）

## 一、项目名称
基于超声的青少年脊柱侧弯筛查与评估系统

## 二、项目定位
本项目面向青少年特发性脊柱侧弯的无创筛查与随访场景，构建基于超声序列的智能分析与评估工作站，实现从图像预处理、骨性结构提取、三维重建到Cobb角候选测量、质量门控与人工复核的全流程闭环系统。

## 三、系统总体架构（符合竞赛模板规范）
系统采用 FastAPI + React + Three.js + SQLite 架构，实现五步工作流：
1. 超声序列预处理
2. 骨性结构提取
3. 冠状面与三维重建
4. Cobb候选计算
5. Reportability Gate与人工复核

## 四、关键技术路线
### 4.1 输入标准化
- 超声序列读取
- 扇形ROI裁剪
- 方向校正与归一化

### 4.2 骨性结构提取
- Classical baseline
- Student v2（部署模型）
- Seg-XL teacher（高精模型）

### 4.3 三维重建
- 冠状MIP生成
- 点云重建
- Three.js可视化

### 4.4 Cobb测量与门控
- 多候选Cobb计算
- boundary-safe策略
- Reportability Gate控制

## 五、实验与测试结果
- 440帧序列端到端验证
- human-line F1@8px = 0.9799
- 147项pytest测试通过
- 五步流程全链路可运行

## 六、创新点
1. 五步可解释医学AI工作流
2. Reportability Gate可信机制
3. 多模型融合Cobb候选系统
4. 人工复核闭环设计
5. 医学软件级审计链路

## 七、结论
系统已从算法原型升级为医学AI工作站雏形，具备可运行、可解释、可追溯的软件级闭环能力，可用于竞赛展示与后续临床验证扩展。
